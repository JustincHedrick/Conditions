<h1>Project Description</h1>
<p>A full-stack MERN web-app that consumes a 3rd party API to quanitify specific weather conditions to best determine overall quality of weather on any given day, for any location in the world.</p>

<h2>Getting Started </h2>
<h3>Instructions </h3>
<details open>
<summary>Accessing Conditions</summary>
<ol>
<li>A user is prompted to search a location using Logitude and Latitude or Log in/Sign up as a new user.</li>
<li>After authorizing, the user is shown the same form to search for a location and any previously saved locations.</li>
</ol>
</details>
<details>
<summary>Using Conditions</summary>
<ol>
<li>After a user or non user has submitted a location with a name, logitude, and latitude weather is then displayed to show the current conditions and quality of day as well as the three day forecast.</li>
</ol>
</details>
<details>
<summary>Create with Conditions</summary> 
<ol>
<li>By clicking the 'save' button, a user can store previously saved locations for later use</li>
 </ol>
</details>
